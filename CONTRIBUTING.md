# Contributing to Thesis Master

Terima kasih udah mau contribute! 🎉

## Cara Contribute

1. **Fork** repo ini
2. Buat **branch** baru: `git checkout -b feature/my-feature`
3. **Commit** perubahan: `git commit -m 'feat: add my feature'`
4. **Push** ke branch: `git push origin feature/my-feature`
5. Buat **Pull Request**

## Bug Reports

Buka [Issues](https://github.com/bayuuuu18/thesis-master/issues) dengan:
- Deskripsi bug
- Steps to reproduce
- Expected vs actual behavior
- Screenshot jika ada

## Feature Requests

Buka [Issues](https://github.com/bayuuuu18/thesis-master/issues) dengan label `enhancement`.

## Code Style

- Python 3.10+
- PEP 8
- Docstrings untuk semua fungsi
- Type hints dimungkinkan

Terima kasih! 🙏
